How do I exclude shared mailbox or users?

Using “Exclude user by csv” file we can exclude both shared mailbox and also regular user using their email id in csv file.

If there are less number o=f shared mailbox (less than 15), you can go for” Exclude Users by Name” option and exclude them by their name.

If there are more number of shared mailboxes you can go for” Exclude User by Using CSV File” option.

    Note:
  1. Export shared mailbox from O365 for email IDs then fill same in CSV file to exclude them.
  2. We can exclude 100 shared mail boxes in one goby CSV file.
Click on upload button of “Exclude User by Using CSV File” row under exclude options tab in settings.


  1. Download the sample file, fill the shared mail box email ID(s) as per specified format and save it, choose the same file and click on Upload button.
