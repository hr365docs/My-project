How do I add Employee Directory Plus as a Web part?

  1. Make sure you installed the Employee Directory Plus in your SharePoint site, then you can add app as web part in same site.

  2. Click on setting gear icon and then Add a page under settings panel.



  3. Choose the Blank page and then click on Create page button.



  4. Add Name, click on round plus icon and then search for Employee Directory Plus and click on it.



  5. Click on Publish button.
