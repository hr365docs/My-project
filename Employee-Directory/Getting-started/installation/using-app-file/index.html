Uploading app file into app catalog

Only SharePoint Administrator can install the appliications. Administrator can make a member an admin.

  1. Go to https://your tenant name-admin.sharepoint.com/_layouts/15/online/TenantAdminApps.aspx.

  2. This displays the Apps page.

  3. Click on App Catalog as shown in below.


  4. Upload

  5. Click on App Catalog as shown in below.

  6. Here we have two option to put app file into the app catalog

  7. Click on upload, choose your file and click on Ok button as shown in below.


  8. Drag your app file from your computer and drop into app catalog.



    9. Installation of App file


  9. Go to Location (i.e., Site collection or subsite) where you want to install the Application and click on setting gear icon.


  10. Click on Add an app



  11. Click on SharePoint Store and type “HR365” in Find an app (search box), all applications from HR365 will sort, now click on application which you want.



  12. Now click on Switch to the classic SharePoint Store


  13. Click on ADD IT button



  14. Note: After clicking of Add IT button, if you find Continue button click it , click on purchase button(this will be a 30 days trial and you don’t need to pay or provide payment detail) your and Return to site button


  15. Click on Trust It


  16. Once you will click on Trust It, application will start installing. Please wait for couple of minutes to complete the installation.


  17. Once it gets completes you will able to click on it.
