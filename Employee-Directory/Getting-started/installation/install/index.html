How to install SharePoint add-in from AppSource and Microsoft store?

  1. Go to Location (i.e., Site collection or subsite) where you want to install the Application and click on setting gear icon.


  2. Click on Add an app



  3. Click on SharePoint Store and type “HR365” in Find an app (search box), all applications from HR365 will sort, now click on application which you want.



  4. Now click on Switch to the classic SharePoint Store


  5. Click on ADD IT button



  6. Note: After clicking of Add IT button, if you find Continue button click it , click on purchase button(this will be a 30 days trial and you don’t need to pay or provide payment detail) your and Return to site button


  7. Click on Trust It


  8. Once you will click on Trust It, application will start installing. Please wait for couple of minutes to complete the installation.


  9. Once it gets completes you will able to click on it.
