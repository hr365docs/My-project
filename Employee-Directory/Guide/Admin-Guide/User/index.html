Users and Roles

    Roles are used to define access rights to modules and determine what a user can do with access to it. There are two roles in Employee director plus:

    1. Admin
    2. User


    Responsibility
    1. Admin
      • Admin has full control.
      • Able to do all settings.
      • Able to edit employees profile.

    2. User
    • User can able to see employee details.
    • Able to edit own profile.
